ℹ️ Project Information

MITS Bank of India Core System

A comprehensive full-stack banking simulation engineered to demonstrate core computer science principles.

Technology Used

  • Node.js & Express.js (Backend Server)
  • MongoDB Atlas (Cloud Database)
  • Object-Oriented Programming (OOPs)
  • HTML5 / CSS3 / JavaScript (Frontend Visualization)

System Capabilities

  • ✅ Create new bank accounts saved securely in the cloud
  • ✅ Deposit and withdraw money with real-time updates
  • ✅ View transaction history with secure ledger timestamps
  • ✅ Search and manage all accounts instantly
  • ✅ Delete account option
